# figma.constants

**Source:** https://developers.figma.com/docs/plugins/api/figma-constants/

---

These are the properties available on the `figma.constants` global object.

### colors: [ColorPalettes](/docs/plugins/api/ColorPalettes/)

Holds color palettes, which maps color names to hex codes.

Example usage

```
const colorMapBase = figma.constants.colors.figJamBase  
const colorMapBaseLight = figma.constants.colors.figJamBaseLight
```
