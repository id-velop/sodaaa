# RemovedNode

**Source:** https://developers.figma.com/docs/plugins/api/RemovedNode/

---

On this page

An representation of a node that has been removed from the document. Can be part of a [DocumentChange](/docs/plugins/api/DocumentChange/).

## Properties[​](#properties "Direct link to Properties")

### removed: true [readonly]

`removed` is set to `true` to distinguish a deleted node from one that is on the document.

---

### type: [NodeType](/docs/plugins/api/nodes/#node-type) [readonly]

The type of the node before it was removed from the document

---

### id: string [readonly]

The id of the node
